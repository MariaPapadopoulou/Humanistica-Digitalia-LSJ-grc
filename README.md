# Who for?

The tool is built in plain **HTML + JavaScript**, requires no installation, and is designed for students, researchers, and teachers of Ancient Greek.


## Features

- Polytonic Ancient Greek input
- Opens the lemma directly in Logeion
- Extremely fast (no dependencies)
- Mobile-friendly layout
- Gentium Plus / Times New Roman typography
- Clean academic interface inspired by *Humanistica Digitalia*

## About Humanistica Digitalia

**Humanistica Digitalia** is a portfolio of digital resources, lexical tools,  
and open editions supporting the study of Ancient Greek language and literature,  
developed and curated by **Dr. Maria Papadopoulou**, University of Crete.

Projects include:

- Digital editions (Theocritus, Diogenes Laertius, etc.)
- TEI/XML text encoding
- Tools for morphology, lexica, and philological annotation
- Knowledge graph prototypes for the ancient world
